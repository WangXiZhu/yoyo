### 不是什么？
	node作为上下文，不是类似于Apache的服务器。它就是一段运行程序

### 阻塞和非阻塞

	非阻塞操作，我们需要使用回调，通过将函数作为参数传递给其他需要花时间做处理的函数

### child_process子进程

	与系统进行交互  类似java中的system模块
	通过系统子进程进行输入和输出
